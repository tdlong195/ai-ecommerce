# AI Commerce Learning Project

## Stack
- Next.js
- TypeScript
- Pages Router
- CSS Modules
- Local mock data first
- No Tailwind CSS

## Goal
Build a small e-commerce website for learning AI coding tools like Codex and Claude Code.

## Coding rules
- Use TypeScript
- Use CSS Modules for styling
- Keep components small and readable
- Do not add unnecessary libraries
- Explain changed files after coding
- Implement one feature at a time
